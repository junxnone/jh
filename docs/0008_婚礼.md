# 婚礼

## 婚礼流程
> 根据各地习俗，婚礼流程可能大不同

- 婚礼筹备
  - 婚礼宾客
  - 婚礼酒店(预约档期)
  - 婚庆公司
  - 婚礼请帖(结婚照)
  - 婚礼车队
  - 接亲流程
    - 摄像
    - 司仪
  - 婚礼用品
    - 新娘手捧花/婚鞋/礼服租用/
    - 新郎西装(西服上衣/西裤/领带/衬衫)
    - 交换对戒
  - 婚房布置
    - 被褥
    - 四件套
    - 装饰品
  - 接亲房(闺房)布置
- 化妆
- 接亲(伴娘拦门/互动游戏/找鞋穿鞋/抱背新娘)
- 中式婚礼
  - 拜天地
  - 改口
- 西式婚礼
- 拍照留念
- 婚宴
- 敬酒
- 散席
